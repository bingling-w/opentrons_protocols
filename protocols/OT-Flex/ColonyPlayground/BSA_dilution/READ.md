# BSA Dilutions in Duplicates (OT-Flex)

## Overview
The protocol performs automated 2 sets of BSA dilution from 24 tubes in tube racks using the Opentrons Flex robot with an 1-channel 1000 µL pipette. The protocol is currently inflexible and only performs the set dilution process displayed below:

## Protocol Summary
- Robot: Opentrons Flex
- Pipette: Flex 1-Channel 1000 µL

Two duplicates of BSA dilutions are created.

*  Reservoir spot A1 will hold liquid for dilution.
*  Tube spot D1 will hold the BSA stock. 
*  Tube spot A1-A3 will hold the first set of BSA dilution.
*  Tube spot B1-B3 will hold the second set of BSA dilution. 
*  Tube spot A6 and B6 will hold the blanks.
